Welcome, catalog helper! 👋

        
This tool adds free YouTube movies to the FlixMine catalog. You need two keys to get started —
        both are free and take about 2 minutes each. Fill them in below, hit Test on each, then Save settings.

    


    

        
TMDb required 

        
How to get a free TMDb key
            
    
                
  1. Create a free account at themoviedb.org
    2. 
                
  2. Go to Settings → API and request a key (choose "Developer", any personal use reason is fine)
    3. 
                
  3. Copy either the API Key or the API Read Access Token — both work here
    4. 
            

        

        

            
            
            
        

    


    

        
Publish target 

        

            
            
        

        

            
How to get a GitHub token (safe, repo-only)
                
    
                    
  1. You need a GitHub account and write access to the catalog repo (ask neeko to add you as a collaborator)
    2. 
                    
  2. Go to github.com → Settings → Developer settings → Fine-grained tokens → Generate new token
    3. 
                    
  3. Under Repository access pick Only select repositories → choose the catalog repo
    4. 
                    
  4. Under Permissions → Repository permissions set Contents to Read and write. Nothing else is needed.
    5. 
                    
  5. Generate and copy the token (starts with github_pat_)
    6.
